# Edutia

## What is it?
EduTIA is an online educational and training gallery that specifically made for Indonesian learner who wants to get ready for the job industry. The courses are specially made for Indonesia job market and requirements. Not only specialize the teaching methods, also EduTIA provides link to job market for the educatee. EduTIA prepare the educatee with skills needed for the companies and help them to find the job that suits their specialty.

## What are its features?
**For Educatees:**
- Learn new skills
- Guided path for specific jobs
- Apply to companies
- Get certification and create a CV

**For Companies:**
- Post job opportunities
- Give job requirements

## Installation
Step:
- npm install
- buat db dengan nama edutia (bebas sebenernya yang penting bikin koneksi)
- cp .env.example .env | kalau butuh aja
- npx prisma migrate dev
- npx prisma db seed
- npm run dev
